Media Queries

MDN - Media Queries 

body {

    background-color: green;

    @media (min-width: 768px) {
        background-color: yellow;
    }

    @media (min-width: 992px) {
        background-color: orange;
    }

    @media (min-width: 1200px) {
        background-color: red;
    }

}
